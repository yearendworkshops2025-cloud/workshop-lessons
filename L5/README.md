# L5: Create Your Own Game!

## What is this lesson about?

Now it's YOUR turn to be creative! Come up with your own game idea and build it with Claude Code!

## The Challenge

1. **Think of a game idea** - ANY game you want!
2. **Describe it to Claude Code**
3. **Build it together**
4. **Debug and improve it**
5. **Share it with others!**

## Game Ideas (if you need inspiration)

- **Quiz Game** - Test knowledge on any topic
- **Memory Match** - Flip cards and find pairs
- **Snake Game** - Classic snake eating food
- **Tic Tac Toe** - Play against the computer
- **Word Guessing** - Like Wordle or Hangman
- **Clicker Game** - Click to earn points and upgrades
- **Maze Game** - Navigate through a maze
- **Rock Paper Scissors** - Play against Claude!

## How to Start

Open your terminal and type:
```
pwd
ls
cd Lessons
cd L5
claude+
```

## Quick Start Command

```
cd ~/Lessons/L5 && claude+
```

Copy this and paste in terminal!

## What to Say to Claude

Describe your game:
> "I want to create a [YOUR GAME IDEA]. Make it playable in the browser with HTML, CSS, and JavaScript. Make it fun and add sound effects or animations if possible!"

## Tips for Success

- Start simple, then add features
- Test often while building
- Ask Claude to explain what the code does
- Don't be afraid to ask for changes!

## What You Will Learn

- How to turn an idea into working code
- How to iterate and improve
- How to work WITH AI as a coding partner
- Creative problem solving

---

**Time estimate:** ~15-20 minutes (depends on complexity)
