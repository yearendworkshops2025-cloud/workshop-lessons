# L9: Spreadsheet Analytics & Dashboards

## What is this lesson about?

Learn to work with **spreadsheet data**! Analyze CSV and Excel files, find patterns, and create beautiful visualizations and dashboards.

## The Challenge

Ask Claude Code to:
1. **Load spreadsheet data** from the `data/` folder
2. **Perform exploratory data analysis** - What's in the data?
3. **Find patterns and insights** - What does the data tell us?
4. **Create visualizations** - Charts and graphs
5. **Build an HTML dashboard** with your findings
6. **Create a presentation** about data analysis

## What You Will Learn

- How to load and read spreadsheet data
- Exploratory Data Analysis (EDA)
- Finding patterns in data
- Creating charts and visualizations
- Building data dashboards

## Key Concepts

- **CSV**: Comma-Separated Values - simple data format
- **XLSX**: Excel spreadsheet format - can have multiple sheets
- **EDA**: Exploratory Data Analysis - understanding data before conclusions
- **Visualization**: Turning numbers into charts and graphs
- **Dashboard**: A page showing multiple data insights

## Available Sample Data

Check the `data/spreadsheets/` folder for:
- **Netflix Movies**: Titles, ratings, years, genres
- **Travel Data**: Cities, distances, modes of transport
- **Sales Data**: Products, quantities, revenue
- And more interesting datasets!

## Dashboard Ideas

- Top 10 lists
- Bar charts and pie charts
- Trends over time
- Category breakdowns
- Key statistics and summaries
