# L10: Content Scraping & Article Creation

## What is this lesson about?

Learn to **gather information from websites**, create briefings, and write complete articles with AI-generated images!

## The Challenge

Ask Claude Code to:
1. **Scrape content** from a website of your choice
2. **Create a "word of mouth" briefing** - Key points someone could share verbally
3. **Write a complete article** based on the scraped content
4. **Generate images** for the article using Gemini Nano Banana
5. **Create a polished HTML page** with the final article

## How to Start

```
cd ~/Lessons/L10 && claude+
```

Copy this and paste in terminal!

## What to Say to Claude
> "Scrape content from [WEBSITE URL]. Create a word-of-mouth briefing with the key points. Then write a complete article about it with AI-generated images using Gemini. Create a beautiful HTML page with the final article!"

## Example Websites to Scrape

- News articles
- Wikipedia pages
- Blog posts
- Product pages
- Company about pages

## What You Will Learn

- How web scraping works
- How to summarize information
- How to create content from research
- Combining text and AI images

## Key Concepts

- **Web Scraping**: Extracting information from websites
- **Briefing**: A summary of key points
- **Content Creation**: Turning information into articles
- **Image Generation**: Creating visuals with AI

## Ethics Note

- Only scrape publicly available content
- Respect robots.txt and terms of service
- Give credit to original sources
- Don't scrape personal data
