# Vibe Coding Workshop - Lessons

Welcome to the **Franz Enzenhofer Vibe Coding Workshop**!

This workshop is designed for **non-technical users** who want to learn coding with the help of Claude Code AI.

---

## How This Works

1. **Go to a lesson folder** (e.g., `cd ~/Lessons/L1`)
2. **Read the README.md** to understand the challenge
3. **Open Claude Code** with `claude++`
4. **Ask Claude to explain** the lesson and create a presentation
5. **Do the challenge** together with Claude
6. **Move to the next lesson!**

## Quick Start Command

```
cd ~/Lessons/L1 && claude
```

Copy this and paste in terminal to start your first lesson!

**Note:** L1 uses `claude` (normal mode). From L2 onwards, use `claude+` (YOLO mode).

---

## Key Commands

| Command | What it does |
|---------|--------------|
| `claude` | Start Claude Code (normal mode) - used in L1 |
| `claude+` | Start Claude Code YOLO mode (skips permissions) - used from L2 onwards |
| `cd Lessons` | Go to Lessons folder |
| `cd L1` | Go to lesson 1 |
| `cat README.md` | Read the lesson instructions |
| `pwd` | Show current folder |
| `ls` | List files in folder |
| **TAB** | Autocomplete! Type `cd L` then press TAB |
