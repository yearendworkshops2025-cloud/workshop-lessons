# L6: Create & Publish a Website!

## What is this lesson about?

Build a **one-page website** and publish it to the internet for everyone to see! Learn about GitHub and web hosting along the way.

## The Challenge

Ask Claude Code to:
1. **Create a one-page website** - About any topic you choose!
2. **Explain what GitHub is** - Where code lives online
3. **Create a GitHub repository** - Store your code in the cloud
4. **Publish to Vercel** - Make it live on the internet!
5. **Create an HTML presentation** about GitHub and publishing

## How to Start

Open your terminal and type:
```
pwd
ls
cd Lessons
cd L6
claude+
```

## Quick Start Command

```
cd ~/Lessons/L6 && claude+
```

Copy this and paste in terminal!

## What to Say to Claude
> "Create a beautiful one-page website about [YOUR TOPIC]. Then explain what GitHub is and create a repository for it. Finally, publish it to Vercel so it's live on the internet. Create an HTML presentation explaining GitHub and web publishing!"

## Website Ideas

- A personal portfolio
- A page about your hobby
- A fan page for something you love
- A recipe collection
- An informational page about a topic

## What You Will Learn

- How to create a website with HTML and CSS
- What GitHub is and why developers use it
- How to publish a website for free
- What web hosting means

## Key Concepts

- **GitHub**: A place to store and share code online
- **Repository**: A folder for your project on GitHub
- **Vercel**: A free platform to host websites
- **Deployment**: Making your website live
